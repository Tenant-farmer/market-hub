### 결정

### 이슈와 해결
| 이슈 | 원인 | 해결 |
|---|---|---|
| 콘솔 출력 크래시 (UnicodeEncodeError) | Windows cp949가 em-dash(—) 인코딩 불가 | print는 ASCII 위주로, 실행 시 `PYTHONIOENCODING=utf-8` |
| 최신일 분석이 2~3개 심볼만 계산됨 | 야후 EOD 확정 지연으로 최신 1~2일이 심볼별로 들쭉날쭉 | `_trim_ragged_tail` — 커버리지 80% 미만 꼬리 날짜 제거. 다음 수집이 자동 보충 |
| 분석 재실행 후에도 옛 날짜 행 잔존 | INSERT OR REPLACE는 삭제를 안 함 | replace_metrics 패턴으로 통일 |
| Stooq 폴백 불가 | JS 봇 검증 도입됨 | 폴백 제거, yfinance 단일 + 실패 격리로 대응 |
| 풋콜비율이 2019년 값 | Cboe equitypc.csv가 레거시 아카이브 | 14일 recency 가드로 스테일 적재만 방지. **현행 소스 교체 미해결** |

| 듀케인 평가액이 천 배 작음 | 일부 제출자가 여전히 천달러 단위 보고 | 포트 총액 < $10M이면 ×1000 보정 |
| KOSDAQ 종목이 코드로 표시 | 업종 구성종목 맵이 KOSPI만 커버 | 수급 수집 시 종목명 캐시(market='KR_NAME', INSERT OR IGNORE — sector_map PK가 stock_code 단독이라 REPLACE 금지) |
| KR 수급 일별 루프 1,000+ 호출 (IP 차단 위험) | 초기 설계 실수 | `get_market_trading_value_by_date` 기간 단일 호출로 교체 |

### 자동 수집 스케줄러 (Windows 작업 스케줄러, 매시간)

- 라우팅: 매번 심리지표 / KR 장중(평일 09:00~16:15) KR 3종+분석 / US 장중(KST 22:30~06:15) 섹터+분석

- 등록: `schtasks /Create /SC HOURLY /TN market-hub-hourly` → run_hourly.bat → data/scheduler.log
- 부수 효과: 장중 실행이 당일 중간 시세를 적재 → 대시보드가 준실시간(1시간 지연)이 됨.

### VIX 매수 타이밍 백테스트 (scripts/vix_backtest.py, 2000~2026 일봉 25년)
- 방법: VIX 구간별 매수→1/3/6개월 성과 + 비중첩 스파이크 에피소드. KOSPI는 전일 VIX 사용(룩어헤드 방지)
- **핵심 발견**
  - 최악의 진입 구간은 VIX 20~25 ("약간 무서울 때"): SPY 6개월 승률 57% < 베이스라인 72%
  - VIX 30~40: SPY 승률 74~79%, 6개월 중앙값 +13.1% — 최고의 상시 구간
  - VIX 40+ 스파이크(18회): SPY 63일 승률 94%, KOSPI 6개월 중앙값 **+31.2%** (승률 93%)
  - 저VIX(<15)는 나쁘지 않음 (126일 84%) — VIX는 매도 신호가 아니라 "공포 매수" 신호
  - 꼬리 주의: 스파이크 진입 최악 -21~-26% (2008처럼 더 떨어질 수 있음 → 분할 필수)
- 캐비앗: 구간 표는 중첩 표본(자기상관), 스파이크 40+는 18회뿐, 비용 미반영, 한 시대(2000~26)
- V-KOSPI(한국판 VIX)는 무료 소스 부재로 미포함 — 대신 미국 VIX가 코스피 타이밍에도 유효함을 확인 (오히려 더 극적)

### VIX × VVIX 결합 백테스트 (scripts/vvix_backtest.py, 2007~2026 SPY 63일)
- **VIX 단독 분석의 "최악 구간"(VIX 20~30)이 VVIX로 갈라짐**:
  VVIX<95면 승률 65%(안일한 하락 = 함정) / VVIX≥95면 **84%·중앙값 +6.9% (전체 최고 셀)**
- 전조 신호: VIX<20 & VVIX≥95 (평온한데 크래시 헤지 수요) → 승률 66~67% (평시 76%보다 나쁨) = 매수 보류
- VVIX 극단(120+) 진입: 중앙값 +6.4%, **최악 -11.5%** (VVIX<85의 최악 -40.3%와 대조 — 공포가 이미 가격에 반영)
- 깊은 공포(VIX≥35)에선 VVIX 냉각(5일평균 아래) 확인 시 중앙값 +5.5%→+9.8%
- VVIX/VIX 비율은 단조 패턴 없음 → 규칙 비채택
- 방법 교훈: 스파이크 "최초 돌파일"엔 VVIX가 항상 상승 중 — 냉각 조건은 일 단위 국면 분석으로
- 실전 규칙(v1): 보류 = [VIX<20 & VVIX≥95] 또는 [VIX 20-30 & VVIX<95] /
  1차 매수 = VIX 20-30 & VVIX≥95 / 분할 확대 = VIX 30+ (35+는 VVIX 냉각 확인 후 추가)
